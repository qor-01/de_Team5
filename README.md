# 프로젝트 설명
**이 프로젝트는 유전체 변이 데이터를 수집하고, 집단 간 차이를 분석하며, 네트워크 모델링을 통해 변이 간의 상호작용을 시각화하는 것을 목표로 합니다. 특히, 기능 예측 점수와 임상적 유의성을 바탕으로 핵심 변이를 식별하고, 의료 및 생명과학 분야에서의 응용 가능성을 탐색합니다.**
